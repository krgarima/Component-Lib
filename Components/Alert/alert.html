Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

1) Simple Alert

Simple Alerts can be created using alerts class, followed by the variant name.

Primary! This is a Primary Alert - Check it now!
Error! This is an Error Alert - Check it now!
Success! This is a Success Alert - Check it now!
Warning! This is a Warning Alert - Check it now!

2) Outlined Alert

Oulined Alerts can be created using alerts class, followed by the variant name.

Primary! This is a Primary Alert - Check it now!
Error! This is an Error Alert - Check it now!
Success! This is a Success Alert - Check it now!
Warning! This is a Warning Alert - Check it now!

3) Actions

An alert can have an action, such as a close or undo button. It is rendered after the message, at the end of the alert.

This is a success alert — check it out!
This is a success alert — check it out!

4) Transition

You can use a transition component such as Collapse to transition the appearance of the alert.
